Ini adalah platform aplikasi Chatbot berbasis website menggunakan Flaskdengan menggunakan metode Deep Learning

Chagubot adalah aplikasi chatbot berbasis website yang dibuat menggunakan teknologi Artificial Intelligence terkhusus domain Natural Language Processing dengan metode Deep Learning model Neural Network. Chagubot bertujuan untuk membantu mengatasi permasalah pelayanan mahasiswa terkait sistem akademik di Jakarta Global University.
